## Description

Nest crud with mongoDB

## Project setup

```bash
$ yarn install
```

```bash
$ npm i -g @nestjs/cli
```

```bash
$ #docker compose up
$ docker compose -f docker-compose.prod.yaml up --build
$ docker compose -f docker-compose.prod.yaml up
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
